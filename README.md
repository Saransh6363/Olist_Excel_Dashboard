# 📊 Olist Excel Dashboard

## 📌 Overview
An interactive Excel dashboard built using the Olist E-commerce dataset to analyze revenue trends, order status distribution, and state-wise sales performance.

---

## 🎯 Objectives
- Analyze revenue generated across customer states.
- Monitor yearly revenue growth.
- Track order fulfillment status.
- Build an interactive dashboard using Excel features.

---

## 📈 Dashboard Features

### KPI Cards
- 📦 Total Orders: 99,441
- 💰 Revenue: ₹1.60 Cr
- 🚚 Delivered Orders: 96,478
- 🌍 Customer States: 27

### Visualizations
- Revenue by State (Top 10 States)
- Orders by Status
- Revenue by Year

### Interactive Elements
- Payment Method Slicer
- Order Status Slicer
- Year Slicer
